# Frontend

## Tecnologias utilizadas
- React
- TypeScript
- Tailwind CSS
- Zustand
- Framer Motion
- React Query
- Vite

## Como executar

```bash
cp .env.example .env.local

# Preencha a variável VITE_API_URL com a URL do backend

npm install
npm run dev
```
O projeto estará disponível em `http://localhost:5173`
